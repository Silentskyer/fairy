# 玩家屬性、境界與貨幣資源系統

## 一、設計目標

本系統要同時滿足 4 件事：

- 支撐文字修仙遊戲的長線成長感
- 讓修練、副本、煉丹、種藥、鍛造彼此連動
- 保留「凡人流」資源稀缺與取捨壓力
- 方便後續轉成資料表、公式與 API

## 二、玩家屬性系統

玩家屬性分成 4 層：基礎屬性、戰鬥屬性、修行屬性、成長標記。

### 1. 基礎屬性

這些是角色最核心的常駐能力。

- `氣血`：角色生命值，歸零即戰鬥失敗
- `靈力`：施展功法、煉丹、鍛造、突破時消耗的主要能量
- `攻擊`：決定普通攻擊與部分術法的基礎傷害
- `防禦`：降低受到的傷害
- `身法`：影響出手速度、閃避與部分探索事件成功率
- `神識`：影響高階功法操控、煉丹穩定度、辨識與感知能力

### 2. 修行資質屬性

這些屬性不一定直接造成戰鬥傷害，但會決定長期效率。

- `悟性`：影響修練效率、功法參悟速度、突破時的額外加成
- `根骨`：影響氣血成長、防禦成長、突破穩定性
- `靈根資質`：影響吐納效率、靈力上限與修練速度
- `丹道資質`：影響煉丹成功率與品質
- `器道資質`：影響鍛造成功率與裝備品質
- `藥植資質`：影響種藥成熟時間、產量與稀有變異機率

### 3. 衍生戰鬥屬性

這些由基礎屬性、境界、裝備、丹藥、功法共同計算。

- `最大氣血`
- `最大靈力`
- `暴擊率`
- `暴擊傷害`
- `命中`
- `閃避`
- `格擋`
- `穿透`
- `抗性`：火、毒、寒、雷等
- `回氣效率`：每回合或每段時間恢復靈力的能力

### 4. 成長標記

這些用來管理長期進度。

- `境界`
- `境界階段`
- `修為`
- `突破進度`
- `聲望`
- `宗門身份`
- `功法熟練度`
- `丹道等級`
- `器道等級`
- `藥植等級`

## 四、屬性作用說明

### 氣血與靈力

- `氣血` 是副本生存底線
- `靈力` 是修士差異的核心資源
- 戰鬥、煉丹、鍛造、突破都會消耗靈力
- 讓玩家必須在「繼續冒險」與「先回復狀態」間做選擇

### 攻擊、防禦、身法

- `攻擊` 決定推圖與打怪效率
- `防禦` 決定容錯率，對凡人流體感很重要
- `身法` 讓玩家能走高輸出低防或穩健生存兩種路線

### 神識、悟性、根骨

- `神識` 偏向中後期，支援煉丹、辨識、特殊副本
- `悟性` 讓修練派、功法派角色更有價值
- `根骨` 讓坦度和突破安全性更有差異

## 五、境界系統

境界是本遊戲的核心成長結構。它不只是數值提升，也會影響：

- 可進入的地圖與副本
- 可學習的功法與丹方
- 可使用的裝備與材料
- 藥田、鍛造、宗門任務的解鎖條件

### 初版建議開放境界

1. 練氣境
2. 築基境
3. 金丹境
4. 元嬰境
5. 化神境

這 5 個大境界可作為目前完整境界骨架；若第一版內容量要控管，可先把主要可遊玩內容做到金丹境或元嬰境，化神境保留作中後期版本目標。

### 每個大境界的內部分層

每個大境界可再拆成 10 個小階：

- 一層到三層：初期
- 四層到六層：中期
- 七層到九層：後期
- 十層：圓滿

這樣好處是：

- 玩家每天都能感受到成長
- 不需要過快開新大境界
- 商店、丹藥、副本可以細緻分段

## 六、境界成長流程

玩家成長建議走這條循環：

1. 修練、戰鬥、服丹取得修為
2. 修為累積到當前小階上限
3. 蒐集突破所需資源
4. 進行突破判定
5. 成功後提升小階或大境界

### 修為來源

- 掛機修練
- 靈石灌注修練
- 副本戰鬥
- 任務獎勵
- 丹藥輔助
- 特殊奇遇

## 七、境界突破與修為需求公式

這套系統以 `修為值` 作為核心成長資源。

### 基本規則

- 最基礎的小境界突破需求為 `100`
- 每突破一個小境界，下一個小境界需求乘以 `1.2`
- 每突破一個大境界，下一個大境界第一層需求乘以 `2`
- 所有需求值一律採 `無條件進位` 處理，避免小數累積

### 公式定義

若目前要從某一層升到下一層，所需修為為：

- 小境界突破：`nextExp = ceil(currentExp × 1.2)`
- 大境界突破：`nextRealmExp = ceil(currentExp × 2)`

### 規則說明

- 同一大境界內，`1 -> 2`、`2 -> 3` ... `9 -> 10` 都算小境界突破
- 每個大境界的 `10層圓滿 -> 下一個大境界1層` 算大境界突破
- 大境界突破完成後，再從新的基礎值開始繼續乘上 `1.2`

## 九、突破條件設計

### 小境界突破條件

- 目前修為值達到需求
- 角色靈力需高於 `70%`
- 角色不可處於重傷、走火入魔等負面狀態
- 消耗少量時間完成突破

### 大境界突破條件

- 當前大境界 `10層圓滿`
- 修為值達到大境界突破需求
- 靈力需高於 `90%`
- 消耗指定突破材料或輔助丹藥
- 需要一次突破判定

### 大境界突破判定建議

大境界突破不建議完全固定成功，可保留一點風險，但不能太重，否則會讓玩家挫折感過高。

建議初版：

- 基礎成功率：`70%`
- 受 `悟性`、`根骨`、丹藥、功法品質影響
- 失敗後不掉大境界，但扣除部分靈力與少量修為

## 十、修練成長方式

你指定的成長方式是：

- 消耗靈石修練
- 功法掛機成長

我將其整理成兩條主要修練路線。

### 1. 功法掛機修練

玩家裝備一門主修功法後，可以持續獲得修為。

#### 建議公式

- `每分鐘修為 = 功法基礎效率 × 功法倍率 × 悟性倍率 × 靈根倍率`

#### 建議初版參數

- 黃階下品功法：`1.0`
- 黃階中品功法：`1.2`
- 黃階上品功法：`1.5`
- 玄階下品功法：`1.8`

#### 屬性倍率建議

- 悟性倍率：`1 + 悟性 / 100`
- 靈根倍率：`1 + 靈根資質 / 100`

#### 掛機限制

- 離線收益上限：`12 小時`
- 靈力不足時，掛機效率下降
- 可透過更換高階功法提升整體效率

### 2. 靈石灌注修練

玩家可選擇直接消耗靈石，加快修為累積速度，對應凡人流裡的資源換時間。

#### 建議公式

- `本次獲得修為 = 消耗靈石 × 靈石轉化率 × 功法倍率`

#### 建議初版參數

- 基礎轉化率：`1 靈石 = 5 修為`
- 若有高品質功法，可提供 `1.1 ~ 1.5` 倍加成
- 每日可加入軟上限，避免課金或囤資源玩家瞬間爆衝

#### 靈石灌注定位

- 適合卡關時快速補足缺口
- 適合突破前最後衝刺
- 會擠壓商店、生產與裝備養成的資源，因此帶來取捨感

## 十一、修練與突破的完整循環

1. 玩家選擇主修功法進行掛機修練
2. 透過掛機、副本、丹藥、任務累積修為
3. 若差少量修為，可消耗靈石快速補足
4. 達到當前層數需求後進行小境界突破
5. 到達 `10層圓滿` 後準備丹藥、材料與狀態
6. 進行大境界突破
7. 解鎖更高等級功法、副本、商店與生產系統

## 十二、數值平衡建議

- 練氣境應讓玩家靠掛機就能穩定成長
- 築基境開始需要副本與丹藥配合
- 金丹境後，靈石灌注只能作補強，不能完全取代其他玩法
- 大境界突破材料要成為副本、商店、煉丹與種藥的交會點
- 高品質功法要明顯提升效率，但不能讓低品質功法完全失去價值

## 十四、戰鬥規則與傷害公式

## 戰鬥定位

戰鬥採 `半回合制`。

它不是傳統你一下我一下的固定輪流，而是以 `身法` 作為速度核心，讓雙方各自推進行動條。誰先把行動條跑滿，誰就先出手。若速度差距過大，可能出現一方連續多次行動，形成壓制。

這種設計適合文字修仙遊戲，因為：

- 能明確體現身法的重要性
- 能產生快攻、控場、風箏、爆發等戰鬥風格
- 能讓高境界或高品質裝備產生更直觀的節奏差

## 行動條規則

### 基本概念

- 每個戰鬥單位都有一條 `行動進度條`
- 進度條滿值為 `1000`
- 戰鬥開始後，雙方依照自身速度持續累積行動值
- 先達到 `1000` 的單位立即獲得一次行動權

### 速度公式

基礎速度由 `身法` 決定。

建議初版：

- `速度值 = 身法`

若之後要擴充，也可以加入：

- 功法速度加成
- 裝備速度加成
- 狀態效果加成或減益

### 行動條推進公式

- `每個計時單位獲得的行動值 = 速度值`

也就是說：

- 身法 `100` 的角色，每 10 個計時單位可行動一次
- 身法 `200` 的角色，每 5 個計時單位可行動一次

### 行動後處理

- 單位出手後，行動條扣除 `1000`
- 若扣除後仍然大於等於 `1000`，代表速度差過大，可以立即再次行動

這正好符合你要的效果：

- 速度快的一方可以先攻
- 速度極高時，可能出現一方持續行動、另一方來不及出手

## 連續行動判定

為了保留你要的壓制感，同時避免前期數值過度失衡，建議規則如下：

### 基本連動機制

- 若 A 的行動條在出手後仍 `>= 1000`，A 立即再次行動
- 若 A 的速度遠高於 B，A 可以在 B 尚未行動前多次出手

### 建議平衡保護

初版可先保留其中一個保護規則：

- 同一單位最多連續行動 `3` 次

這不是硬性必要，但若沒有上限，某些高身法 build 可能直接把低速怪打到完全沒互動。

若你想要更殘酷一點，也可以後續把這個保護拔掉。

## 行動內容

每次獲得行動權後，單位可執行以下其中一種操作：

- 普通攻擊
- 施放功法
- 使用丹藥或道具
- 進入防禦或蓄力狀態
- 施加增益或減益

初版最基本可先實作：

- 普通攻擊
- 功法攻擊
- 使用丹藥

## 傷害公式

你指定的基礎傷害公式為：

- `基礎傷害 = 攻擊 × 2 - 防禦`

我把它整理為正式規則如下。

### 1. 普通攻擊基礎傷害

- `baseDamage = 攻擊方攻擊 × 2 - 受擊方防禦`

### 2. 最低傷害保護

為避免高防目標讓傷害變成負數或 0，建議加上最低保護值：

- `baseDamage = max(1, 攻擊方攻擊 × 2 - 受擊方防禦)`

### 3. 功法傷害

功法在基礎傷害之上，再乘上功法倍率：

- `skillDamage = baseDamage × 功法倍率`

建議初版功法倍率範例：

- 低階攻擊功法：`1.2`
- 中階攻擊功法：`1.5`
- 高爆發功法：`1.8 ~ 2.2`

### 4. 屬性克制

你指定的五行克制為：

- 金剋木
- 木剋土
- 土剋水
- 水剋火
- 火剋金

若攻擊屬性克制防禦屬性，傷害提高 `1.5` 倍。

公式：

- `finalDamage = skillDamage × 1.5`

若無克制，則維持原值。

### 5. 最終建議公式

初版可以統一寫成：

- `finalDamage = max(1, 攻擊 × 2 - 防禦) × 功法倍率 × 屬性克制倍率`

其中：

- `功法倍率` 預設為 `1`
- `屬性克制倍率` 預設為 `1`
- 若發生克制則 `屬性克制倍率 = 1.5`

## 命中與閃避建議

你目前沒有指定命中閃避公式，所以初版建議先簡化：

- 普通攻擊與功法默認命中
- 後續若要加入身法對戰鬥更深影響，再額外設計 `命中 - 閃避`

這樣能先把速度條與傷害主體跑起來，不會一次把戰鬥系統做得太重。

## 戰鬥平衡建議

- 身法很重要，但不能完全壓過攻擊與防禦
- 前期怪物速度差不要太大，避免新手被連動壓死
- 中後期再逐步放出高身法怪物與高速流派
- 五行克制倍率 `1.5` 已很有感，初版不建議再疊額外克制加成
- 高防角色仍需吃到最低 1 點傷害，避免無限僵局

## 十五、各境界的體感定位

### 練氣境

- 新手期
- 資源極度匱乏
- 以生存、打基礎、熟悉系統為主
- 可接觸低階煉丹、種藥與鍛造

### 築基境

- 進入真正修士世界
- 可接觸宗門試煉與較穩定的資源點
- 商店與副本逐漸成體系

### 金丹境

- 角色開始形成明確流派
- 功法、裝備、丹藥的搭配性提升
- 生產玩法收益顯著增加

### 元嬰境

- 中後期核心目標
- 解鎖高階坊市、稀有丹方、特殊洞府
- 可作為版本第一個大里程碑

### 化神境

- 高階版本目標
- 接觸世界真相、上古遺跡與飛升線索
- 解鎖頂級資源爭奪與高風險副本

## 十六、建議初版屬性成長邏輯

### 境界提升時固定成長

- `maxHp` 穩定提升
- `maxQi` 大幅提升
- `attack` 與 `defense` 小幅提升
- `spirit` 依中後期逐步拉高

### 裝備與丹藥提供短中期差異

- 裝備決定副本能力與鍛造循環
- 丹藥決定修練效率、突破成功率與戰鬥續航
- 功法決定流派特色

### 生活技能提供生產差異

- `alchemySkill` 影響煉丹收益
- `forgingSkill` 影響裝備產出價值
- `herbalismSkill` 影響種藥效率與原料成本

## 十七、貨幣系統

貨幣系統建議採多貨幣分層，但初版只先實作 3 個核心貨幣。

### 1. 靈石

主貨幣，也是整個經濟系統的基礎流通單位。

用途：

- 商店購買
- 裝備修理
- 部分副本門票
- 藥田開墾
- 鍛造爐租用

來源：

- 副本掉落
- 任務獎勵
- 出售材料
- 種藥與煉丹成品販售

### 2. 宗門貢獻

宗門相關貨幣，用於換取內部資源。

用途：

- 宗門商店兌換
- 特定功法、丹方、法器圖譜
- 宗門藥田、煉丹房、地火房使用資格

來源：

- 宗門任務
- 宗門試煉
- 上繳材料

### 3. 黑市令牌

高風險貨幣，對應灰色勢力與稀有交易。

用途：

- 購買違禁材料
- 換取稀有秘境情報
- 取得高風險高收益商品

來源：

- 黑市任務
- 特殊副本
- 事件奇遇

## 二十、核心經濟循環

### 戰鬥循環

副本戰鬥 -> 掉落材料 / 靈石 -> 強化角色 -> 挑戰更高副本

### 生產循環

種藥 / 採藥 -> 煉丹 -> 修練 / 販售 -> 換取更多資源

### 鍛造循環

挖礦 / 副本掉礦 -> 鍛造裝備 -> 提升推圖能力 -> 獲得更高價值材料

### 商店循環

販售多餘資源 -> 獲得靈石 -> 補足短缺材料 -> 提升生產與戰鬥效率

這 4 個循環應互相咬合，避免任何單一玩法獨立成島。

## 二十一、經濟設計原則

- 靈石要重要，但不能萬能
- 部分高階資源必須透過副本、任務或聲望解鎖
- 玩家可以靠副本致富，也可以靠生產致富
- 商店負責補缺，不應完全取代玩家自己生產
- 種藥、煉丹、鍛造應形成「省成本 + 提效率 + 可交易」的長期價值
